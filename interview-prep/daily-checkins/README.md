# 28 天打卡总表

1. 每天只打开当天文件，不要来回切。
2. 先完成 `必做`，有余力再做 `加分项`。
3. 晚上必须完成当日交付物，不然第二天不算开始。
4. 每周最后一天先复盘，再进入下一周。
