# Java Backend Interview Prep Pack

这套资料是按你当前的目标定制的：

- 目标岗位：中小厂 Java 后端
- 准备周期：1 个月
- 项目主线：直接打磨 `LifeOS` 微服务项目
- 核心原则：先能投，再做深

建议使用顺序：

1. 先看 [00-glossary.md](./00-glossary.md)，把常见缩写和名词先过一遍。
2. 再看 [01-4-week-roadmap.md](./01-4-week-roadmap.md)，按周推进。
3. 再看 [02-lifeos-project-breakdown.md](./02-lifeos-project-breakdown.md)，把项目链路和亮点讲顺。
4. 接着背 [03-project-scripts.md](./03-project-scripts.md)，先形成可输出版本。
5. 日常执行直接打开 [daily-checkins/README.md](./daily-checkins/README.md)，按天打卡。
6. 每天配合 [04-high-frequency-qa.md](./04-high-frequency-qa.md) 复盘口述。
7. 算法按 [05-algorithm-plan.md](./05-algorithm-plan.md) 执行。
8. 简历和模拟面试按 [06-resume-and-mock.md](./06-resume-and-mock.md) 收口。
9. 如果你想自己加练，再用 [07-daily-template.md](./07-daily-template.md) 补充。

你这一轮准备最重要的不是“学完所有东西”，而是做到三件事：

- 能把 `登录鉴权 -> 网关转发 -> 业务服务 -> MySQL/Redis -> 返回结果` 讲明白
- 能把 `Redis / JWT / RocketMQ / Nacos` 为什么用、怎么用、出了问题怎么办讲清楚
- 能在 30 分钟面试里稳定输出，不被自己的项目细节问倒

推荐日配比：

- `1.5h` 技术专题
- `1.5h` 结合项目定位代码入口和整理笔记
- `1h` 高频八股口述
- `0.5-1h` 算法
- `0.5h` 复盘

如果你不想自己安排顺序，直接从 `daily-checkins/day-01.md` 开始，每天完成一页即可。
